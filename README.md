# OrangeHRM SaaS Admin E2E Automation Framework

End-to-end test automation framework built using **Playwright + TypeScript**.

## 🚀 Project Goal

Automate a SaaS-style admin dashboard (OrangeHRM demo) using:

- Page Object Model
- Smoke test strategy
- CI/CD pipeline (GitHub Actions)
- Clean TypeScript architecture

## 🧪 Current Coverage

- Login → Dashboard (Smoke)
- Logout flow

## 🛠 Tech Stack

- Playwright
- TypeScript
- GitHub Actions

## ▶ How to Run Locally

```bash
npm install
npm run test:smoke
